    # Input输入框

    WARNING

    Input 为受控组件，它总会显示 Vue 绑定值。

    通常情况下，应当处理 input 事件，并更新组件的绑定值（或使用v-model）。否则，输入框内显示的值将不会改变。

    不支持 v-model 修饰符。

    # Input Attributes

    参数 说明 类型 可选值 默认值
    type number:小数点或者小数点后两位 integer:只能输入整数 text:任意类型 String number, integer, text string
    point 小数点后位数 Number 2, 3 2

    TIP

    其他参数和element-ui类似，可以直接用element-ui参数

    # Input Methods

    方法名 说明 参数
    change 输入框失去焦点
    search 用户按下回车时触发或者点击搜索图标

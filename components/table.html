    # Table Attributes

    TIP

    其他参数和element-ui类似，可以直接用element-ui参数

    参数 说明 类型 可选值 默认值
    tableData 表格数据 Array
    tableColumn 表格参数对象 Array
    rank 是否显示排行 String 排行
    bitSet 是否显示操作行 Boolean false
    btnFixed 操作列是否固定在左侧或者右侧，true 表示固定在左侧 String, Boolean true, false, 'left', 'right' false
    btnWidth 操作行的宽度 String 120px
    totalElements 总条目数, 如果大于设置的pageSize则显示分页 Number
    pageSize 每页显示条目个数 Number 10

    # TableColumn Attributes

    参数 说明 类型 可选值 默认值
    width 对应列的宽度 String
    minWidth 对应列的最小宽度，与 width 的区别是 width 是固定的，min-width 会把剩余宽度按比例分配给设置了 min-width 的列 String
    sortable 对应列是否可以排序 Boolean
    toolTip moneyType 当内容过长被隐藏时显示 tooltip Boolean 默认隐藏显示，如果为true，不隐藏 false
    moneyType 是否做金额处理 Boolean 小数点后两位 false
    unit 显示数据单位 String 商品价格可以显示单位，例如：10/公斤
    custom 是否自定义列 String

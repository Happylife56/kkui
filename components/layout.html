    # 基础用法

    常用的外边距设置

    mt10 上边距10px：margin-top:10px
    mr10 右边距10px：margin-right:10px
    mb10 下边距10px：margin-bottom:10px
    ml10 左边距10px：margin-left:10px
    mt20 上边距20px：margin-top:20px
    mr20 右边距20px：margin-right:20px

    # 样式属性class

    方法名 说明 参数
    mt10, mt20 上边距10px，上边距20px
    mb10, mb20 下边距10px，下边距20px
    ml10, mt20 左边距10px，左边距20px
    mr10, mr20 右边距10px，右边距10px
    line-through 中划线
    pointer 光标手指指针
    not-allowed 禁用样式
    flex 设置flex布局
    flex-warp 超出换行
    flex-align 左右居中
    flex-justify 上下居中
    flex-center 左右上下居中
    flex-around 两侧的间隔相等
    flex-between 两端对齐，项目之间的间隔都相等
    flexs-justify 多轴方式：左右居中
    flexs-align 多轴方式：上下居中
    flexs-center 多轴方式：上下左右居中
